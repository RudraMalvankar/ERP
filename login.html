Login Help

For Students

  • Use your student email address (e.g., john.doe@student.college.edu)
  • Your initial password is usually your student ID
  • Select "Student" from the user type dropdown
  • Contact IT support if you've forgotten your credentials

For Staff/Faculty

  • Use your institutional email address
  • Your password was provided during onboarding
  • Select "Staff/Faculty" from the user type dropdown
  • Contact HR for password reset assistance

For Administrators

  • Use your admin email address
  • Ensure you have proper administrative privileges
  • Select "Administrator" from the user type dropdown
  • Contact system administrator for technical issues
